# How to:
* Clone the repository
* Run <code>npm i</code>
* Run <code>git checkout</code>
* Inspect logs written to your terminal window
* Inspect **package.json** -> "husky"->"hooks"
* Check out [Git Hooks docs](https://githooks.com) for more info
* Play around
* That's it :) Enjoy!
